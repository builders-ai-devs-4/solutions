Use this tool to gather all information needed to plan a route to Skolwin.
The Explorer will run at least $MAX_SEARCH_ITERATIONS search queries to discover
available data sources, then fetch map, vehicles, terrain rules and positions.
Returns a structured report with all gathered data.

Pass a task description in natural language. Include:
- the goal (reaching Skolwin)
- what data you need (map, vehicles, terrain rules, starting position)

Call this tool ONCE before calling the planner.
